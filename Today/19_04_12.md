## docker save
```sh
$ docker save busybox > busybox.tar
$ ls -sh busybox.tar
2.7M busybox.tar

$ docker save --output busybox.tar busybox
$ ls -sh busybox.tar
2.7M busybox.tar

$ docker save -o fedora-all.tar fedora
$ docker save -o fedora-latest.tar fedora:latest
```
